# Архитектура VBA Generator

## Компонентная схема

```text
[ Пользователь / Browser ]
          |
          v
[ Frontend (React) ]
  - client/src/App.tsx
  - client/src/pages/Home.tsx
  - TanStack Query (useQuery/useMutation)
          |
          | HTTP
          v
[ Backend API (Express) ]
  - server/index.ts
  - server/routes.ts
  - POST /api/generate
  - GET  /api/history
  - GET  /api/examples
       /         \
      v           v
[ Anthropic ]   [ SQLite + Drizzle ]
  Claude API      - server/storage.ts
  JSON result     - shared/schema.ts
                   - table: generations
      \           /
       v         v
  [ API response payload ]
  { id, code, explanation, category, steps }
          |
          v
[ Frontend Result UI ]
  - отображение кода VBA
  - explanation / steps
  - история и примеры
```

## Поток генерации

1. Пользователь вводит задачу в `Home.tsx`.
2. Клиент отправляет `POST /api/generate`.
3. `routes.ts` валидирует вход и вызывает Claude.
4. Ответ Claude парсится как JSON.
5. Генерация сохраняется в SQLite через `storage.ts`.
6. API возвращает результат на фронтенд.
7. UI показывает VBA-код, пояснение и шаги установки.
